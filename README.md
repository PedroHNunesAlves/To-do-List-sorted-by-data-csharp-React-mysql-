# Instruções

- Baixar os modules e iniciar pastas nodebackend e frontend. 

1. Na pasta "nodebackend", executar:
   ```
   npm start
   ```

2. Na pasta frontend, executar os seguintes comandos:
  ```
  cd app
  ```
  ```
  npm run dev
  ```
   
## PS.:
